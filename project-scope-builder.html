Bug Bounty Scope Builder

Web application that automatically builds a structured bug bounty scope for a company. It discovers official mobile apps, identifies likely public APIs and generates a professional scope document in minutes.

Reduces manual reconnaissance effort, improves scope accuracy and uses AI to enhance results.

  • Mobile Apps: When a company website is entered, official Apple App Store and Google Play apps are automatically discovered and included in the generated scope.
  • API Discovery: If a company has a public API, the web is searched for likely endpoints and the strongest candidates are passed to AI to identify the most probable official API.
  • Editor: Allows bold, italic and structured formatting so the final scope reads like a professional document rather than plain text.
  • Auto Save: Saves user input automatically to prevent lost work.
  • Configurable: Scope text and reward tiers can be changed easily without impacting the program.
  • Reliability: Built-in error handling so the program continues working safely if an external service fails.

Technical Information

  • Integrations: Connects to a custom API that retrieves data and includes an AI recommendation automatically in the generated scope.
  • Modules: 5 separate modules instead of one large file, making the system easier to maintain and extend.
  • Deterministic scope generation: Builds the scope using defined templates and rules, keeping output consistent and repeatable.
  • Config-driven behaviour: Scope content, rewards and discovery rules are stored in JSON so changes can be made safely without rewriting core logic.
  • Controlled AI usage: Uses AI only where reasoning adds value (recommendations), reducing speculative output and keeping results reliable.
  • Resilient by design: If external services fail, the UI handles errors cleanly and continues operating safely without breaking the user flow.
  • State-first UX: Automatically saves progress to prevent lost work and reduce user frustration during longer scope builds.
  • Codebase: 15 organised source files, approximately 3.7k lines of code.
