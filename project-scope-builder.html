Bug Bounty Scope Builder

Web application that retrieves mobile apps and APIs using a custom-built API, then constructs a bug bounty program scope from the URL, discovered assets and selected rewards.

Reduces manual reconnaissance effort, produces more accurate scopes and demonstrates disciplined AI-assisted development with strong architectural control.

  • Modules: 5 separate modules instead of one large file, making the system easier to maintain and extend.
  • Config-driven: Key content and rules stored in 3 JSON files so updates can be made without rewriting core code.
  • Integrations: Connects to a custom API that retrieves data and includes an AI recommendation automatically in the generated scope.
  • Reliability: Built-in error handling so the program continues working safely if an external service fails.
  • State management: Saves user input automatically to prevent lost work.
  • Editor: Allows bold, italic and structured formatting so the final scope reads like a professional document rather than plain text.
  • Codebase: 15 organised source files, approximately 3.7k lines of code.
