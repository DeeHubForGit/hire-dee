Bug Bounty Scope Builder API

Backend API that provides verified asset intelligence to support accurate scope definition for bug bounties.

  • Mobile app discovery: Searches the Apple App Store and Google Play to retrieve official mobile apps for a company (e.g. “Coles”), including related apps (e.g. “Coles Mobile”). Allows search by app name, developer name, either (broad search) or both (narrow search) to control how wide the results are.
  • API discovery: Searches the web for likely API names associated with a company domain, applies rule-based filtering to narrow results, then uses AI to select the most probable official API endpoint.
  • Input validation and clear errors: Validates requests and returns helpful error messages when something is missing or invalid, instead of failing silently.
  • Built-in documentation: Includes self-serve API documentation pages (Swagger UI and ReDoc) so the service is easier to understand, test and maintain.

Technical Information

  • Deterministic discovery first: Asset retrieval follows clear, rule-based logic before any AI involvement, ensuring consistent and repeatable results.
  • Reliable asset intelligence: Returns clear, consistent results so scope decisions are based on verified information rather than guesswork.
  • Controlled AI usage: AI is applied only where reasoning adds value, reducing the risk of inaccurate or speculative results.
  • Consistent mobile discovery: Shared logic across iOS and Android keeps behaviour aligned and reduces duplication, improving maintainability and reliability.
  • Config-driven behaviour: Discovery rules and thresholds are configurable, allowing safe extension of the system without breaking existing logic.
  • Codebase: 5 organised source files, approximately 2.8k lines of code.
