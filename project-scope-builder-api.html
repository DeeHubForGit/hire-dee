🔌 Bug Bounty Scope Builder API

Backend API for discovering mobile applications and API assets associated with a company domain.

Designed to support bug bounty and security programs by automating asset discovery and returning deterministic asset data for scope definition.

Backend API AI Integration System Architecture Security

What It Does

  • Retrieves mobile applications (iOS and Android) associated with a company
  • Identifies API endpoints and related assets for a given URL
  • Returns structured, explainable asset data suitable for downstream scope definition

Note: This API does not generate scope documents. It provides the asset intelligence needed to define scope accurately.

Key Technical Decisions

  • Deterministic discovery first: Asset retrieval uses explicit logic and heuristics before any AI involvement
  • Controlled AI usage: AI is used only where reasoning is required, not for guessing or discovery
  • Shared mobile discovery architecture: iOS and Android logic implemented via a shared delegate pattern to reduce duplication and keep behaviour consistent
  • Config-driven behaviour: Discovery rules and thresholds are configurable, allowing safe extension without breaking existing logic
